## 🤖 點樣使用 AI Agent

**步驟：**
1. 打開你想傾嘅教練文件（例如 `Alex_Hormozi.md`）
2. 搵到 `## 🤖 AI Agent System Prompt` 嗰段
3. 複製 ` ``` ` 入面嘅所有內容
4. 打開 **Claude App** → 開新對話
5. 貼入去對話嘅**第一句**
6. 然後問你嘅問題 — AI 會以該教練身份回答你

**例子：**
```
[貼入 Alex Hormozi 的 System Prompt]

你：我而家想設計一個針對痛症客戶嘅 IG offer，你覺得我應該點寫？

AI：*以 Alex Hormozi 身份回答* ...
```
